# 1.2.4 NPM和Node安装
NPM : Node Package Manager
在确保node.exe和git都在PATH环境变量中后执行以下命令:
    
1. `git clone --recursive git://github.com/isaacs/npm.git
`
2. `git clone --recursive git://github.com/isaacs/npm.git`
3. `cd npm`
4.  `node cli.js install npm -gf`
